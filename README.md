# OpenGL Project

/////////OVERVIEW////////////
C/C++ project using openGL and SDL libraries. I used Eclipse CDT. 

Main goal is to escape in the UFO by jumping over the boxes.

//////CONTROLS/////////
z : up
q : left
d: right
s: down
space : jump
Use the mouse to move
