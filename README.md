# OpenGL Project

## Overview
C/C++ project using openGL 1.2 and SDL 2  
Main goal is(was) to escape in the UFO by jumping over the boxes.

## Controls
z : up  
q : left  
d: right  
s: down  
space : jump  
Use the mouse to move  

## Build
### Prerequisite
MinGW

### Build command
mingw32-make

## Launching the game
Release/game.exe
